Last-Edit-Finder:
Last-Edit-Finder is a lightweight browser extension designed for students, researchers, and fact-checkers. It instantly identifies when a website was last modified, helping you determine the "currency" and reliability of your sources at a glance.

Why Use Last-Edit-Finder?: In academic research, the CRAAP Test (Currency, Relevance, Authority, Accuracy, and Purpose) is vital. Many websites hide their publication dates or make them hard to find. This extension automates that search by:

- Scanning Meta Tags for hidden timestamps.

- Providing a quick way to verify if information is outdated.

Features:

- One-Click Check: View the last modified date directly from your browser toolbar.
- Smart Detection: Looks for Last-Modified headers and article:modified_time tags.
- Minimalist Design: No heavy UIâ€”just the information you need, when you need it.
- Student-Focused: Built specifically to assist with citations and source evaluation.

Installation (Manual/Developer Mode)

You can install it manually:
Download or clone this repository: git clone https://github.com/Guptan0506/Last-Edit-Finder.git
Open your browser and go to the Extensions page:
Chrome: chrome://extensions/
Edge: edge://extensions/
Toggle on Developer mode (usually in the top right).
Click Load unpacked and select the folder containing the extension files.

How to Use

  Navigate to any website you are using for research.
  Click the Last-Edit-Finder icon in your extension bar.
  The popup will display the most accurate "Last Modified" date found for that specific page.
